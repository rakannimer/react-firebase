# Setup

Installation

#### API

```bash
yarn add generate-json
```

#### CLI

```bash
# In your project :
yarn add -D generate-json
yarn generate-json <schemaFilePath>

# Or temporarily
npx generate-json <schemaFilePath>

# Or globally :
npm i -g generate-json
generate-json <schemaFilePath>
```
