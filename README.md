## Installation

### Web

Install [firebase](https://www.npmjs.com/package/firebase).

```bash
yarn add firebase
# Or 
npm i firebase
```

#### Firebase Auth

```bash
yarn add @react-firebase/auth
```

#### Firebase Realtime Database

```bash
yarn add @react-firebase/database
```

#### Firebase Firestore Database

```bash
yarn add @react-firebase/firestore
```
